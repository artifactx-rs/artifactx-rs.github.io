ArtifactX

Build Once. Package Once. Publish Everywhere. v0.1.0

# 1 minute from zero to signed repo
arx init                         # create repo + key
arx pack ./Cargo.toml            # build .deb .rpm .apk
arx publish                      # sign + index
arx serve                        # HTTP on :8080
Source Download Docs Docker

What

One static binary that replaces aptly, reprepro, createrepo, and nfpm — apt + yum repository management, pure-Rust packaging, built-in HTTP serving, with PGP signing and atomic rollback. No database, no daemon, no platform.

Install

curl -fsSLO https://github.com/artifactx-rs/artifactx/releases/download/v0.1.0/arx-amd64
sudo install -m 755 arx-amd64 /usr/local/bin/arx

Or cargo build --release · docker run ghcr.io/artifactx-rs/arx:v0.1.0 serve

Migrate

arx import http://your-old-repo/apt --apt        # from existing apt repo
arx import http://your-old-repo/yum/myrepo --yum  # from existing yum repo
arx publish                                          # one click, done
