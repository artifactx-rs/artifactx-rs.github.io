ArtifactX

Build Once. Package Once. Publish Everywhere. v0.1.0

Source Download Docs Docker 
# 1 minute from zero to signed repo
arx init                         # create repo + key
arx pack ./Cargo.toml            # build .deb .rpm .apk
arx publish                      # sign + index
arx serve                        # HTTP on :8080

What

One static binary replacing aptly, reprepro, createrepo, and nfpm — apt + yum repo management, pure-Rust packaging, built-in HTTP serving, PGP signing, atomic rollback. No database, no daemon.

Install

curl -fsSLO https://github.com/artifactx-rs/artifactx/releases/download/v0.1.0/arx-amd64
sudo install -m 755 arx-amd64 /usr/local/bin/arx

Also: cargo build --release · docker run ghcr.io/artifactx-rs/arx:v0.1.0 serve

Migrate

Pull packages from an existing apt or yum repo:

arx import https://old-repo.example.com/apt --apt
arx publish
