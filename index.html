ArtifactX

One static binary. apt + yum repo management. Pure-Rust packaging. v0.1.0

GitHub Download Docs Docker

Migrate from any existing repo — one command

Import packages from Ubuntu, Docker CE, ClickHouse, or your own apt/yum repo. No directory listing needed — parses metadata exactly like apt-get and dnf do.

arx import https://packages.clickhouse.com/deb --apt --match-name clickhouse
arx import http://archive.ubuntu.com/ubuntu --apt --dist focal --limit 20
arx publish                     # one click, done — your own signed repo
# 1 minute from zero
arx init                         # create repo + key
arx pack ./Cargo.toml            # build .deb .rpm .apk
arx publish                      # sign + index
arx serve                        # HTTP on :8080

Install

curl -fsSLO https://github.com/artifactx-rs/artifactx/releases/download/v0.1.0/arx-amd64
sudo install -m 755 arx-amd64 /usr/local/bin/arx

or docker run ghcr.io/artifactx-rs/arx:v0.1.0 serve · cargo build --release
