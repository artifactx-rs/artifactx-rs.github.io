Why ArtifactX

The missing middle between scripts and artifact platforms.

Keep aptly-style safety, nfpm-style packaging, and Cloudsmith-style push workflows without inheriting a database, JVM, Django stack, SaaS bill, or packaging toolchain.

Package

One manifest to native packages

Build .deb, .rpm, and .apk from Rust code without shelling out to dpkg-deb, rpmbuild, FPM, or nfpm.

Publish

Signed metadata, atomic states

Generate Debian Release/InRelease and RPM repodata, sign them, then flip published states atomically.

Operate

Rollback, GC, import, serve

Remove bad packages, prune old versions, import existing apt/yum repos, and expose a compatible HTTP server from the same binary.

Install

Start from a binary, container, or source checkout.

All releases

Static binary

curl -fsSLO https://github.com/artifactx-rs/artifactx/releases/download/v0.1.0/arx-amd64
sudo install -m 755 arx-amd64 /usr/local/bin/arx
arx --version

Docker

docker run --rm -v $(pwd)/repo:/repo -p 8080:8080 \
  ghcr.io/artifactx-rs/arx:v0.1.0 serve \
  --root /repo --addr 0.0.0.0:8080

Docker Compose

services:
  arx:
    image: ghcr.io/artifactx-rs/arx:v0.1.0
    ports: ["8080:8080"]
    volumes: ["./repo:/repo"]

Build from source

git clone https://github.com/artifactx-rs/artifactx.git
cd artifactx
cargo build --release
./target/release/arx --version

Migrate

Import an existing repo, then publish your own signed copy.

ArtifactX parses metadata the way package managers do, so you can pull from Ubuntu, Docker CE, ClickHouse, or a private apt/yum repository without relying on directory listings.

arx import https://packages.clickhouse.com/deb --apt --match-name clickhouse
arx import http://archive.ubuntu.com/ubuntu --apt --dist focal --limit 20
arx publish
Command Purpose
arx initScaffold a repository and generate a PGP signing key.
arx packBuild native packages from a Cargo.toml or manifest.
arx publishSign and index apt/yum metadata with atomic staging.
arx serveRun an HTTP server compatible with apt, dnf, REST API clients, and metrics scraping.
arx pushUpload packages to a remote arx serve with OIDC or static token auth.
arx rm / gcYank packages and prune old versions with EVR-aware retention.
arx rollbackFlip back to a previous published state.
arx importMigrate packages from an existing apt/yum repository.
