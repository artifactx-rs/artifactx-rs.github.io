ArtifactX

Build Once. Package Once. Publish Everywhere.

One static binary. apt + yum repository manager. Pure-Rust packager. Zero dependencies.

GitHub Download apt repo 
# 1 minute from zero to signed repo
arx init                              # create repo + signing key
arx pack ./Cargo.toml                 # build .deb .rpm .apk
arx publish                           # sign + index
arx serve                             # HTTP server on :8080

What it does

📦 Repository

Generate signed apt + yum repos. Atomic publish with symlink flip. Rollback in one command.

🔨 Pack

Pure-Rust .deb / .rpm / .apk builder. Cargo.toml-driven. Reproducible builds.

🚀 Publish

Serve over HTTP. Push via OIDC (no stored token). REST API for CI integration.

⚡ Fast

Incremental publish — O(changes), not O(repo). Version-aware GC. Static binary.

🔐 Secure

PGP signing. Valid-Until freeze protection. Key rotation. OIDC keyless push.

🪶 Simple

One binary. No database. No daemon. Backup with tar. Restore and republish.

curl -fsSLO https://github.com/artifactx-rs/artifactx/releases/download/v0.1.0/arx-amd64
sudo install -m 755 arx-amd64 /usr/local/bin/arx
